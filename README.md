# Phishing Please: An Educational Security Awareness Game

This is the official implementation for the educational project: "Phishing Please: An Immersive Security Awareness Simulation for Enhancing Threat Identification in Corporate Environments."

This application provides an interactive, gamified approach to cybersecurity training. It challenges users to navigate 7 distinct progressive scenarios designed to bridge the gap between theoretical knowledge and practical defense against social engineering and phishing tactics.

## Download

The latest stable version for Windows is available for download in the releases section.

## Key Features

7 Progressive Scenarios: Features a curriculum-based level design ranging from identifying basic email red flags to detecting advanced social engineering techniques.
Interactive Training Environment: Includes a dedicated "Basic Training" module to onboard users with the mechanics of digital threat identification.
Realistic Simulations: Level designs are modeled after common professional digital communication interfaces to improve knowledge transfer to real-world situations.
Immediate Feedback System: Provides real-time explanations for user choices, reinforcing correct security behaviors through active learning.
High-Fidelity Rendering: Utilizes Unity's Universal Render Pipeline (URP) to provide a clear, professional visual experience optimized for desktop environments.

## Getting Started

To get a local copy up and running for development, follow these steps.

### Prerequisites

Unity Hub
Unity Editor 2022.3.x (with Windows Build Support)

### Installation

1. Clone the repository:
   git clone https://github.com/reiidj/Phishing-Please.git
2. Open the project in Unity Hub.
3. Wait for the initial import to complete (this may take several minutes as Unity rebuilds the Library folder).
4. Build and run the application:
   File > Build and Run
